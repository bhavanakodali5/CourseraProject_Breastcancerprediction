# CourseraProject_Breastcancerprediction

## Breast Cancer Detection

This project builds a Breast Cancer Detection system using Logistic Regression in Python. The model is trained on the Breast Cancer Wisconsin dataset to classify tumors as **malignant (cancerous)** or **benign (non-cancerous)** based on medical features. The workflow includes data preprocessing, label encoding, feature scaling, train-test splitting, model training, and evaluation using accuracy score and confusion matrix. The results show that Logistic Regression performs effectively in predicting breast cancer with high accuracy.
